Project: pourtoi.love

Stack:
- Laravel backend (API)
- Next.js frontend
- Mobile API via api.pourtoi.love (same Laravel app)

Domains:
- Web: pourtoi.love and www.pourtoi.love
- API: api.pourtoi.love and www.api.pourtoi.love
- WebSocket: ws.pourtoi.love

Paths:
- API root in Nginx: /var/www/backend/public (see nginx/api.conf)

Ports:
- Next.js: 3000 (see nginx/web.conf)
- Reverb: 8080 (localhost only, see nginx/ws.conf)
