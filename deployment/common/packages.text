Packages to install (Ubuntu 22.04 LTS or 24.04 LTS)

Baseline tools:
- git
- curl
- wget
- unzip
- zip
- composer
- ca-certificates
- gnupg
- lsb-release
- software-properties-common
- build-essential

Web + TLS:
- nginx
- certbot
- python3-certbot-nginx

Database:
- mysql-server

PHP 8.4 (standard):
- Add repository first: `sudo add-apt-repository -y ppa:ondrej/php && sudo apt update`
- php8.4-fpm
- php8.4-cli
- php8.4-common
- php8.4-mysql
- php8.4-xml
- php8.4-mbstring
- php8.4-curl
- php8.4-zip
- php8.4-bcmath
- php8.4-intl
- php8.4-gd
- php8.4-opcache

Node.js:
- nodejs + npm via NodeSource or nvm (recommended)

Process manager:
- supervisor

Optional (use only if needed):
- redis-server
- memcached
- imagemagick
- php8.4-imagick
